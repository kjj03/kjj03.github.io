Deploy Manual

이 사이트를 외부에 독립 배포하고 공유하는 법

이 웹 서비스는 단 하나의 HTML 파일로 완결 설계되어 있어, 타인에게 무료로 쉽고 가볍게 호스팅하여 서비스할 수 있습니다.

1. GitHub Pages로 1분 무료 호스팅하기

  1. 전체 HTML 코드를 복사하여 내 PC에 index.html 이름의 파일로 저장합니다.
  2. 본인의 깃허브(GitHub) 계정에 새 레포지토리(Repository)를 개설합니다.
  3. 방금 저장한 index.html 파일을 해당 저장소에 그대로 업로드합니다.
  4. 저장소 설정(Settings) -> Pages 탭에서 소스를 Deploy from a branch로 선택하고 브랜치를 main(혹은 master)으로 저장합니다.
  5. 잠시 후 부여되는 고유 github.io 주소 링크로 친구들과 함께 접속하면 실시간 작동이 개시됩니다!

2. 외부 구글 로그인 불가 오류 해결하기 (중요)

본인의 Firebase 계정을 연동해 구글 계정 동기화를 외부 도메인에 적용하고 싶다면, 파이어베이스 콘솔(console.firebase.google.com) -> Authentication -> Settings -> Authorized Domains 목록에 본인이 배포한 사이트 도메인(예: my-site.github.io)을 직접 추가해 줘야 구글 인증 거부 에러가 나지 않고 정상 동작합니다.
